Mars Introduction

Overview

Sunny TOF camera can provide the point cloud data and depth data via USB port.

TOF SDK provides an interface to communicate with Sunny TOF camera to get the point cloud/depth data and control the camera operation mode. The functions of the SDK:

  • Get the depth data via USB to display the point cloud image and pseudo color image;
  • Capture the depth images and video.
  • Use mouse to rotate and zoom the point cloud images.
  • Control the TOF camera working mode: Mono-frequency or Multi-frequencies;
  • Control the frame rate of the TOF camera;
  • Control the exposure time of the TOF camera.

Working platform

Windows:

  • Windows 7 and up

Linux:

  • Ubuntu14.04 64bit

  • Libusb
